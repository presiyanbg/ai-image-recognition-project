## Installation

Clone GIT repository 

```bash
git clone https://github.com/presiyanbg/ai-image-recognition-project.git
```

## Set up Anaconda

```
Update Environments index

Create Environment with Python 3.7

	Install Spyder

	Install Flask

	Install Keras

	Install Pillow

	Install Tensorflow


Launch Spyder

	Launch web_server.py
	Launch training.py
```
